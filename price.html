Server in Deutschland
100% DSGVO-konform
Hierfür haben wir drei Preismodelle ins Leben gerufen:
Projektbasiertes Modell
Sie zahlen einen Betrag für das gesamte Projekt.
Die Anzahl der Nutzer ist bei dieser Variante unerheblich für den Preis.
Nutzerbasiertes Modell
Sie zahlen einen Betrag pro Nutzer.
Für jeden weiteren Nutzer ist der Erwerb einer weiteren Nutzerlizenz erforderlich
Enterprise Modell
Sie erhalten sämtliche Vorteile des projektbasierten Modells und zusätzlich noch weitere Vorzüge:
  • Unbegrenzter Speicherplatz
  • Hosting über eigene Server optional
  • VR Software (ALLVR)
